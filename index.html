MERGED PULL REQUESTS

PR-1

Added the solution code in C++ of the question SAVE THE PRISONER from HackerRank

In this question candies have to be distributed to every prisoner who are seated in a circular manner .The last candy is a sour one. We have to return the prisoner number who gets the sour candy.

The time complexity involved here is O(1).

I have also updated the code link in the README.md file

PR-2

Added the solution code in java of the question CANDIES from HackerRank

In this question Alice wants to give candies to her children. Every children has a rating. Each child should get atleast one candy and if two are sitting together the one with higher rating should get more number of candies. Minimizing the number of candies required, we have to return the total candies required.

The time complexity involved here is O(n)

I have also updated the code link in the README.md file
